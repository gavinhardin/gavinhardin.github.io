Predicting Kickstarter Success with Ensemble Classfication

  • Tools: Python, Sklearn, Pandas, Gensim, Keras, Matplotlib
  • Concepts: Natural Language Processing, Neural Networks, Discretization, Data Processing

Leading a small group of peers, we proposed and developed a heterogeneous ensemble classifier to predict the success of a given Kickstarter project using natural language processing (NLP) and trained neural networks. We used Doc2Vec to generate feature vectors for projects' names, which were then used by a logistic regression classifier to predict the success category a project would fall under. Other attributes of the project (e.g., fundraising time, goal amount, project category) were fed into a multilayer perceptron to generate another prediction of success, and then weighted probability averaging was performed to generate a final prediction using the predictions of each separate classifier.
